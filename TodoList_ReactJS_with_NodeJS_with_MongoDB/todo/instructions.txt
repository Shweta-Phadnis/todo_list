

To Do List - React & Node JS 

This app helps to add a todolist , mark a task as active / completed and show the total count of tasks 

To run the app : 

Pull the code and create a folder in IDE 
Go to the path in which you have the package.json 
npm install 
npm start 

The app will will be available in localhost and default port 3000 
If node server runs in default port then run the client in different server
